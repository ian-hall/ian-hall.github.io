Some Current Projects

fitness log

a project i did in college that has been reimplemented using the latest microsoft technologies. you will need to register an account to do most stuff, but I promise not to sell your e-mail. this is hosted using a shared instance on microsoft azure which can take some time to spin up if no one has accessed the site in a while (which means always). here are some highlights:

  • create and update exercise logs
  • create and update food logs
  • create reports based on exercise or food logs
  • tools to calculate your BMI and RMR
  • responsive layout powered by twitter bootstrap

Elite: Dangerous Rare Routes

A project I am doing to mess around with Python. Reads data from the spreadsheet found here and throws everything into a genetic algorithm to find viable routes for trading rare goods in the game Elite: Dangerous.

I also did a thing to generate silly rare goods that you can find here.

Cellular Automata simulator

I like me some cellular automata so I wrote this to play around with various types of 1D and 2D automata. Supports multiple rule sets and user created populations.
