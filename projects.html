Some Current Projects

fitness log

a project I'm doing to learn ASP.NET MVC. you will need to register an account to do most stuff, but I promise not to sell your e-mail. this is hosted using a shared instance on microsoft azure which can take some time to spin up if no one has accessed the site in a while (which means always). here are some highlights:

  • create and update exercise logs
  • create and update food logs
  • create reports based on exercise or food logs
  • tools to calculate your BMI and RMR
  • responsive layout powered by twitter bootstrap

Elite: Dangerous Rare Routes

A project I did/am doing to learn Python. Reads data from the spreadsheet found here and throws everything into a genetic algorithm to find viable routes for trading rare goods in the game Elite: Dangerous.

Conway's Game Of Life

I like me some cellular automata so I wrote this to play around with Conway's Game Of Life. What originally started as a basic Java program written for CS101 many moons ago has turned into this...thing written in C#.
